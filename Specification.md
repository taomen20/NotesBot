Нужно разработать чат бот для Телеграм, который будет собирать записки у пользователей, оплачивать их через платежный шлюз и по запросу чтеца выводить для прочтения. После прочтения отправитель будет получать подтверждение прочтения.
Проект нужно реализовать на Pyton.
Записки до момента прочтения должны храниться в базе данных.
Взаимодействие с Телеграмм через web-hook.
